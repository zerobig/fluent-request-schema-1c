# Установка

Библиотека "Fluent запросы" поставляется в двух вариантах. Как конфигурация и как расширение. В случае если вы хотите внедрить библиотеку в свою конфигурацию, то вам необходимо провести объединнение своей конфигурации с файлом поставки конфигурации. Если вы желаете использовать расширение, то просто скачайте его и добавьте в соответствующем разделе.

## Требования

* 1С Предприятие версии 8.3.9 и выше
